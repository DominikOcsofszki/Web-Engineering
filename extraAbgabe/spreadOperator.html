Spread Operator

The spread operator is a useful feature in JavaScript that allows you to distribute elements of an array. This makes it easy to combine, copy, or extend arrays and objects. The spread operator is represented by three dots (...).

Copying Arrays




            const originalArray = [1, 2, 3];
            const copiedArray = [...originalArray];

            console.log(copiedArray); // Output: [1, 2, 3]

Copying Object Properties


            const table = [...document.querySelectorAll("table")].pop();
            const mapSortValues = [...wordMap].sort((b, a) => a[1] - b[1]);
            const obj1 = { foo: 'bar' };
            const obj2 = { ...obj1 };

            console.log(obj2); // Output: { foo: 'bar' }
